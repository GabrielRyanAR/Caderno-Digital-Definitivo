Aula 2 - 2º Trimestre - Mais Portas Lógicas

Hoje vimos mais portas lógicas:

Porta NAND: A porta NAND (Not AND) é uma combinação das portas AND e NOT. Ela produz uma saída de 0 somente se todas as entradas forem 1. Se qualquer uma das entradas for 0, a saída será 1. Em termos simples, é o oposto da porta AND. A tabela verdade para a porta NAND é a seguinte:

  • Entrada: 0, 0 | Saída: 1
  • Entrada: 0, 1 | Saída: 1
  • Entrada: 1, 0 | Saída: 1
  • Entrada: 1, 1 | Saída: 0

    • Porta NOR: A porta NOR (Not OR) é uma combinação das portas OR e NOT. Ela produz uma saída de 1 somente se todas as entradas forem 0. Se qualquer uma das entradas for 1, a saída será 0. É, essencialmente, o oposto da porta OR. A tabela verdade para a porta NOR é a seguinte:

  • Entrada: 0, 0 | Saída: 1
  • Entrada: 0, 1 | Saída: 0
  • Entrada: 1, 0 | Saída: 0
  • Entrada: 1, 1 | Saída: 0

    • Ambas as portas NAND e NOR são conhecidas como "portas universais" porque qualquer outra porta lógica (AND, OR, NOT) pode ser criada usando apenas portas NAND ou apenas portas NOR. Isso as torna extremamente versáteis no design de circuitos digitais.
